Installation & démarrage

Prérequis

  • Python 3.10+, MySQL 8.0
  • Dépendances : pip install -r requirements.txt
  • Variables dans .env : connexion MySQL, clé secrète JWT

Lancement

  1. Créer la base ged_pme et configurer .env
  2. Exécuter python app.py depuis backend-python/
  3. Ouvrir http://localhost:5000
  4. Premier compte : inscription entreprise (/register) ou admin global pré-configuré

Les sauvegardes SQL sont stockées dans backend-python/backups/.

Interface Employé

Accès : /dashboard-employee après connexion avec le rôle employé.

Workflow documentaire

  1. Créer un document (brouillon) avec titre, catégorie et fichier
  2. Soumettre pour validation par l'admin PME
  3. Suivre le statut : en révision → validé/rejeté → publié
  4. Consulter les documents publiés de l'entreprise

Fonctions principales

  • Aperçu : PDF, images, DOCX, XLSX, CSV
  • Versions : historique et restauration de vos documents
  • OCR : extraction de texte pour recherche
  • Partage : email, WhatsApp, QR code
  • Favoris et recherche avancée
  • Profil : mot de passe, alertes WhatsApp, 2FA

Interface Admin PME

Accès : /dashboard-pme — gestion d'une entreprise.

Validation

Onglet Validation : approuver ou rejeter (avec commentaire) les documents soumis par les employés.

Publication directe

Quand l'admin PME publie un document, celui-ci est immédiatement validé et visible par les employés (sans passer par le workflow employé).

Gestion

  • Employés : création, activation/désactivation
  • Catégories : organisation documentaire, partage par email
  • Corbeille : restauration ou suppression définitive
  • Paramètres : identité entreprise (NIF, RCCM, logo), sauvegarde SQL, configuration workflow

Workflow configurable

Dans Paramètres → Configuration du workflow : définissez les étapes (nom, rôle responsable, délai en heures). Exemple : Soumission (employé, 24 h) → Validation (admin PME, 48 h).

Documents

Recherche multi-critères, aperçu tous formats, Versions et Journal d'activité par document.

Interface Admin global

Accès : /dashboard-admin-global — supervision de toute la plateforme.

Onboarding PME

Créer une entreprise avec NIF, RCCM, logo et compte admin PME (pas un second admin global).

Supervision

  • Vue documents toutes entreprises (lecture, aperçu, téléchargement)
  • Gestion utilisateurs et entreprises (CRUD)
  • Onglet En attente : consultation seule (pas de validation)
  • Activité : rapport par utilisateur + export CSV
  • Paramètres plateforme : maintenance, WhatsApp, email
  • Sauvegardes : liste et téléchargement SQL

Sécurité & double authentification

  • Authentification JWT (session 24 h)
  • Mots de passe hashés (bcrypt)
  • Isolation des données par entreprise
  • 2FA TOTP : disponible dans le profil (employé, admin PME, admin global)

Activer la 2FA

  1. Profil → Configurer la 2FA
  2. Scanner le QR code avec Google Authenticator ou Microsoft Authenticator
  3. Saisir le code à 6 chiffres et activer
  4. À la connexion, un second code sera demandé

FAQ

Je ne vois pas les documents d'une autre entreprise

Normal : chaque PME est isolée. Seul l'admin global voit toutes les entreprises.

Format de fichier non prévisualisable

Utilisez Télécharger ou lancez l'OCR si c'est une image/PDF scanné.

WhatsApp ne fonctionne pas

Configurez CallMeBot dans le profil (clé API personnelle) et vérifiez le numéro au format international (+243…).

Mode maintenance

L'admin global peut activer la maintenance ; seuls les admins globaux conservent l'accès.
